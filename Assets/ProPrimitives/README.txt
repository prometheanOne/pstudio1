Version 1.03
DESCRIPTION:
ProPrimitives package includes all the scripts required to create parametric primitives as Cylinders
spheres, cubes, planes and capsules just from a click away.

Also primitives can be called from a simple API if you want to create the primitives via scripting.



USAGE:
To open the menu window for ProPrimitives go to 
Window --> ProPrimitives

To find how to create primitives from code, refer to "Example.cs"

For more detailed info on how to use this package, you can also check:
http://www.youtube.com/watch?v=qC5j0OQ25qo
